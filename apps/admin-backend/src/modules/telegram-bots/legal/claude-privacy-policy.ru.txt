ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ 
сервиса «Claude официальная подписка»

Настоящая Политика конфиденциальности описывает, какие данные могут обрабатываться при использовании Telegram-бота @claudeaioffibot, а также в каких целях такие данные используются.

Сервис «Claude официальная подписка» уважает право пользователей на конфиденциальность и принимает разумные технические и организационные меры для защиты пользовательских данных.

1. КАКИЕ ДАННЫЕ МОГУТ ОБРАБАТЫВАТЬСЯ

При использовании Telegram-бота @claudeaioffibot могут обрабатываться следующие данные:

1.1. Telegram ID пользователя.

1.2. Username / никнейм Telegram, если он указан в профиле пользователя.

1.3. Данные заказа:
- номер заказа;
- выбранный товар или тариф;
- сумма заказа;
- дата и время оформления;
- статус оплаты;
- статус выполнения заказа;
- результат активации.

1.4. Данные, необходимые для обработки обращения в поддержку:
- описание проблемы;
- сообщения пользователя;
- скриншоты, видео или иные материалы, добровольно предоставленные пользователем.

1.5. Технические данные, необходимые для обеспечения работы сервиса, предотвращения злоупотреблений и проверки заказов.

2. ЦЕЛИ ОБРАБОТКИ ДАННЫХ

Данные пользователей могут использоваться для следующих целей:

2.1. Оформление и обработка заказов.

2.2. Предоставление цифровых товаров и результатов активации.

2.3. Проверка статуса оплаты.

2.4. Работа службы поддержки.

2.5. Предотвращение мошенничества, спама, злоупотреблений и несанкционированных действий.

2.6. Улучшение стабильности и качества работы Telegram-бота.

2.7. Исполнение обязательств перед пользователем в рамках публичной оферты.

3. ПЕРЕДАЧА ДАННЫХ ТРЕТЬИМ ЛИЦАМ

3.1. Данные могут передаваться платежным системам, банкам, криптовалютным платежным сервисам или иным платежным посредникам исключительно в объеме, необходимом для обработки платежа.

3.2. Данные могут передаваться техническим сервисам, используемым для обеспечения работы Telegram-бота, хранения заказов, обработки платежей и поддержки пользователей.

3.3. Сервис не продаёт персональные данные пользователей третьим лицам.

3.4. Данные могут быть раскрыты в случаях, предусмотренных применимым законодательством, при наличии законного требования уполномоченных органов.

4. ХРАНЕНИЕ ДАННЫХ

4.1. Данные хранятся в течение времени, необходимого для выполнения заказа, оказания поддержки, предотвращения злоупотреблений и исполнения обязательств перед пользователем.

4.2. Данные заказов могут храниться после выполнения заказа для подтверждения факта оплаты, передачи цифрового товара, рассмотрения обращений и разрешения спорных ситуаций.

4.3. Пользователь может обратиться в поддержку через Telegram-бот с запросом по обработке своих данных.

5. БЕЗОПАСНОСТЬ ДАННЫХ

5.1. Сервис принимает разумные технические и организационные меры для защиты данных пользователей от несанкционированного доступа, изменения, раскрытия или уничтожения.

5.2. Доступ к данным ограничивается только теми лицами и системами, которым он необходим для обработки заказов, платежей и обращений в поддержку.

5.3. Сервис не запрашивает лишние данные, если они не требуются для выполнения заказа или рассмотрения обращения.

6. ДАННЫЕ ПЛАТЕЖЕЙ

6.1. Оплата может осуществляться через сторонние платежные системы.

6.2. Сервис не хранит полные данные банковских карт пользователей.
