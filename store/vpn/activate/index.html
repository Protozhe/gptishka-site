Выберите систему

Подключение на Windows

  1. Скачайте и установите рекомендованное приложение для подключения.
  2. Откройте приложение и выберите импорт конфигурации по ссылке или ключу.
  3. Вставьте ваш персональный ключ доступа.
  4. Сохраните подключение.
  5. Нажмите кнопку подключения.
  6. После успешного подключения ваш интернет-трафик будет идти через VPN.
Если приложение попросит разрешение на создание VPN-подключения, подтвердите его.

Подключение на Android

  1. Установите приложение для подключения VPN из магазина приложений.
  2. Откройте приложение и выберите импорт по ключу или ссылке.
  3. Вставьте выданный вам ключ доступа.
  4. Сохраните конфигурацию.
  5. Включите подключение.
  6. Подтвердите системный запрос на создание VPN.
При первом запуске Android может показать предупреждение о VPN-подключении — это нормально.

Подключение на iPhone / iPad

  1. Установите подходящее приложение для VPN-подключения.
  2. Откройте его и выберите добавление конфигурации по ключу или ссылке.
  3. Вставьте ваш персональный ключ.
  4. Сохраните подключение.
  5. Нажмите подключиться.
  6. Подтвердите системное разрешение на добавление VPN-конфигурации.
iOS может запросить ввод пароля устройства для подтверждения установки VPN-профиля.

Важно знать перед подключением

  • Ваш ключ персональный и привязан к одной системе.
  • Не передавайте его другим людям.
  • Не используйте ключ одновременно на нескольких устройствах.
  • При смене устройства может потребоваться перевыпуск доступа через поддержку.

Не получается подключиться?

Если приложение не принимает ключ или VPN не запускается, обратитесь в поддержку GPTishka и приложите скриншот ошибки.
